Guidelines

Do's and Don'ts

Do
  • Use Inter with cv01 and ss03 on all text
  • Use JetBrains Mono for all financial amounts with tnum
  • Use weight 510 as default emphasis
  • Use gold as the sole chromatic accent
  • Use luminance stepping for depth
  • Use semi-transparent white borders
  • Use gold focus ring for keyboard accessibility
Don't
  • Use pure white (#FFF) as primary text
  • Use heavy shadows for elevation
  • Apply gold decoratively
  • Use positive letter-spacing on display text
  • Use weight 700 (bold) — maximum is 590
  • Introduce additional chromatic colors
  • Use zebra striping in tables
